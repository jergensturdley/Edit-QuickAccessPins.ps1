# Edit-QuickAccessPins.ps1

A PowerShell script with a simple GUI for quickly viewing and removing pinned items from Windows Explorer Quick Access.

## Usage

1. Run `Edit-QuickAccessPins.ps1` in PowerShell.
2. Select the pinned items you want to remove.
3. Apply changes and reopen File Explorer to see the updated Quick Access list.

## Requirements

- Windows 10 or later
- PowerShell 5.1 or later
